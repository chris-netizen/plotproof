# PlotProof — Flutter app

The PlotProof mobile/web client. See the [project README](../README.md) for what PlotProof is,
the live contract address, and full architecture.

## Quick start

```bash
flutter pub get
flutter run                # on a connected device
flutter build apk --release
```

Source lives in [`lib/`](lib/): `geocell.dart`, `evidence.dart`, `services/`, `screens/`.
